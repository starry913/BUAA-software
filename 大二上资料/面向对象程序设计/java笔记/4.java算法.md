## 1. List（列表）- 动态数组

### 什么是 List？
List 是一个有序的集合，可以存储重复的元素。最常用的是 `ArrayList`。

### 基本操作

#### 创建 List
```java
// 创建空列表
List<Integer> list = new ArrayList<>();

// 创建并初始化
List<String> list2 = new ArrayList<>(Arrays.asList("a", "b", "c"));
```

#### 添加元素
```java
List<Integer> list = new ArrayList<>();

list.add(1);        // 在末尾添加
list.add(2);
list.add(0, 10);    // 在索引0处插入10，后面的元素自动后移
// 结果：[10, 1, 2]
```

#### 获取元素
```java
int value = list.get(0);    // 获取索引0的元素
int size = list.size();     // 获取列表长度
```

#### 删除元素
```java
list.remove(0);                    // 删除索引0的元素
list.remove(Integer.valueOf(2));  // 删除值为2的元素（找到第一个）
```

#### 查找和判断
```java
boolean contains = list.contains(1);  // 判断是否包含1
int index = list.indexOf(1);          // 返回第一个匹配的索引，不存在返回-1
int lastIndex = list.lastIndexOf(1);  // 返回最后一个匹配的索引
```

#### 修改元素
```java
list.set(0, 100);  // 将索引0的元素改为100
```

#### 遍历 List
```java
// 方式1：普通for循环
for (int i = 0; i < list.size(); i++) {
    System.out.println(list.get(i));
}

// 方式2：增强for循环（推荐）
for (Integer num : list) {
    System.out.println(num);
}
```

#### List 排序
```java
// 升序排序
Collections.sort(list);

// 降序排序
Collections.sort(list, Collections.reverseOrder());

// 自定义排序（Lambda表达式）
Collections.sort(list, (a, b) -> a - b);  // 升序
Collections.sort(list, (a, b) -> b - a);  // 降序
```

#### List 转数组
```java
// List 转数组
Integer[] arr = list.toArray(new Integer[0]);

// 数组转 List
List<Integer> list = new ArrayList<>(Arrays.asList(1, 2, 3));
```

#### 清空和判断
```java
list.clear();           // 清空所有元素
boolean isEmpty = list.isEmpty();  // 判断是否为空
```

## 2. Set（集合）- 去重集合

### 什么是 Set？
Set 是一个不包含重复元素的集合。最常用的是 `HashSet`。

### 基本操作

#### 创建 Set
```java
// 创建空集合
Set<Integer> set = new HashSet<>();

// 创建并初始化
Set<String> set2 = new HashSet<>(Arrays.asList("a", "b", "c"));
```

#### 添加元素
```java
Set<Integer> set = new HashSet<>();

set.add(1);
set.add(2);
set.add(1);  // 重复元素不会添加，set中还是只有{1, 2}
```

#### 删除元素
```java
set.remove(1);  // 删除元素1
```

#### 查找和判断
```java
boolean contains = set.contains(1);  // 判断是否包含1
boolean isEmpty = set.isEmpty();     // 判断是否为空
int size = set.size();                // 获取集合大小
```

#### 遍历 Set
```java
// 增强for循环
for (Integer num : set) {
    System.out.println(num);
}
```

#### Set 转 List
```java
List<Integer> list = new ArrayList<>(set);
```

#### 并集、交集、差集
```java
Set<Integer> union = new HashSet<>(setA); // 先复制 setA
union.addAll(setB); //A∪B
union.retainAll(setB);//A∩B
union.removeAll(setB);//A-B

```

#### 清空
```java
set.clear();  // 清空所有元素
```

## 3. Map（映射）- 键值对

### 什么是 Map？
Map 存储键值对（key-value），一个键对应一个值，键不能重复。最常用的是 `HashMap`。

### 基本操作

#### 创建 Map
```java
// 创建空映射
Map<String, Integer> map = new HashMap<>();
```

#### 添加/更新元素
```java
map.put("apple", 1);   // 添加键值对
map.put("banana", 2);
map.put("apple", 3);   // 更新已存在的key，值变成3
```

#### 获取元素
```java
Integer value = map.get("apple");              // 获取"apple"对应的值，不存在返回null
Integer value2 = map.getOrDefault("orange", 0);  // 不存在返回默认值0
```

#### 删除元素
```java
map.remove("apple");  // 删除key为"apple"的键值对
```

#### 查找和判断
```java
boolean containsKey = map.containsKey("apple");  // 判断是否包含某个key
boolean containsValue = map.containsValue(1);    // 判断是否包含某个value
boolean isEmpty = map.isEmpty();                 // 判断是否为空
int size = map.size();                           // 获取映射大小
```

#### 遍历 Map
```java
// 方式1：遍历所有key
for (String key : map.keySet()) {
    System.out.println(key + ": " + map.get(key));
}

// 方式2：遍历键值对（推荐，效率更高）
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}

// 方式3：只遍历值
for (Integer value : map.values()) {
    System.out.println(value);
}
```

#### 获取所有键或值
```java
Set<String> keys = map.keySet();        // 获取所有键
Collection<Integer> values = map.values();  // 获取所有值
```

#### 清空
```java
map.clear();  // 清空所有键值对
```

## 4. 常用应用场景

### 统计字符出现次数（使用 Map）
```java
Scanner sc=new Scanner(System.in);
String str=sc.nextLine();

Map<Character,Integer> map=new HashMap<>();
for(char c:str.toCharArray())
{
    map.put(c, map.getOrDefault(c,0)+1);
}

//按key升序排序
List<Character> list1=new ArrayList<>(map.keySet());
Collections.sort(list1);
for(char m:list1)
{
    System.out.println(m+":"+map.get(m));
}

//按value降序排序
List<Map.Entry<Character,Integer>> list2=new ArrayList<>(map.entrySet());
list2.sort((e1,e2)-> e2.getValue()-e1.getValue());

for(Map.Entry<Character,Integer> m:list2)
{
    System.out.println(m.getKey()+":"+m.getValue());
}
```
### 大整数乘法
```java
import java.math.BigInteger;
import java.util.*;   // 引入集合相关类（List, Map, Set）

public class t1 {
    public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();

        BigInteger ans=BigInteger.valueOf(1);

        for(int i=1;i<=n;i++)
        {
            ans=ans.multiply(BigInteger.valueOf(i));
        }
        System.out.println(ans);


    }
}
```

## 5. 三者对比

| 特性 | List | Set | Map |
|------|------|-----|-----|
| 是否有序 | ✅ 有顺序 | ❌ 无序（或自动排序） | ❌ 无序（或自动排序） |
| 是否可重复 | ✅ 可以重复 | ❌ 不能重复 | 键不能重复，值可以重复 |
| 存储方式 | 单个元素 | 单个元素 | 键值对（key-value） |
| 通过索引访问 | ✅ 可以（list.get(0)） | ❌ 不可以 | ❌ 通过键访问（map.get("key")） |
| 常用实现 | ArrayList | HashSet | HashMap |
| 适用场景 | 需要顺序、允许重复 | 需要去重 | 需要通过键快速找值 |

### 简单记忆方法

- **List**：像数组，有顺序，可以重复 → 用 `ArrayList`
- **Set**：去重用的，不能重复 → 用 `HashSet`
- **Map**：像字典，通过键找值 → 用 `HashMap`
