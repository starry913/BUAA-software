## 4. 使用set避免重复访问（vis数组优化）

**问题场景**：在BFS、优先队列等算法中，需要记录已访问的状态，避免重复处理。

**传统方法的问题**：
- 使用二维数组 `int vis[N][N]` 会占用 O(N²) 空间
- 当 N 很大（如 10⁵）时，数组大小会达到 10¹⁰，导致内存溢出

**优化方法：使用 `set<pair<int, int>>`**

### 基本用法

```cpp
#include <bits/stdc++.h>
using namespace std;

set<pair<int, int> > vis;  // 记录已访问的状态

// 检查是否已访问
if (vis.find({x, y}) == vis.end()) {
    // 未访问，进行处理
    vis.insert({x, y});  // 标记为已访问
    // ... 其他操作
}
```

### 完整示例：多路归并问题

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    long long sum;
    int i, j;
};

struct Compare {
    bool operator()(Node a, Node b) {
        return a.sum > b.sum;  // 小顶堆
    }
};

priority_queue<Node, vector<Node>, Compare> pq;
set<pair<int, int> > vis;

int main() {
    // ... 初始化数据
    
    // 初始元素入队并标记
    pq.push({a[1] + b[1], 1, 1});
    vis.insert({1, 1});  // 在添加前就标记，避免重复添加
    
    while (!pq.empty()) {
        Node top = pq.top();
        pq.pop();
        
        // 处理当前元素
        printf("%lld ", top.sum);
        
        // 添加相邻元素（在添加前检查并标记）
        if (top.i < N && vis.find({top.i + 1, top.j}) == vis.end()) {
            vis.insert({top.i + 1, top.j});  // 先标记
            pq.push({a[top.i + 1] + b[top.j], top.i + 1, top.j});
        }
        if (top.j < N && vis.find({top.i, top.j + 1}) == vis.end()) {
            vis.insert({top.i, top.j + 1});  // 先标记
            pq.push({a[top.i] + b[top.j + 1], top.i, top.j + 1});
        }
    }
    
    return 0;
}
```

### 关键要点

1. **在添加前标记**：在将元素加入队列前就标记为已访问，避免同一元素被多次加入队列
   ```cpp
   if (vis.find({x, y}) == vis.end()) {
       vis.insert({x, y});  // 先标记
       pq.push({...});      // 再入队
   }
   ```

2. **初始元素也要标记**：初始元素在入队前就要标记
   ```cpp
   pq.push({初始值, x, y});
   vis.insert({x, y});  // 立即标记
   ```

3. **检查方法**：使用 `vis.find({x, y}) == vis.end()` 判断是否未访问

### 复杂度分析

- **空间复杂度**：O(k)，其中 k 是实际访问的状态数，而不是 O(N²)
- **时间复杂度**：每次查找和插入为 O(log k)
- **优势**：当实际访问的状态数远小于总状态数时，节省大量空间

### 适用场景

- BFS/DFS 中记录已访问节点
- 优先队列/BFS 中避免重复入队
- 状态空间很大但实际访问状态较少的情况
- 需要记录二维坐标对 (x, y) 的访问情况

### 注意事项

1. **pair 的比较**：`set<pair<int, int>>` 会自动按字典序排序，无需自定义比较函数
2. **内存优势**：当 N=10⁵ 时，数组需要 400GB，而 set 只需要存储实际访问的状态
3. **性能**：虽然查找是 O(log k)，但在实际应用中通常比大数组更高效

### 字符串查找

```cpp
string str = "Hello World Hello";

// 查找子串
str.find("World");             // 返回第一次出现的位置，未找到返回string::npos
str.find("World", 5);          // 从位置5开始查找
str.rfind("Hello");            // 从后往前查找，返回最后一次出现的位置

// 查找字符
str.find('o');                 // 查找字符'o'第一次出现的位置
str.find_first_of("aeiou");    // 查找第一个出现在"aeiou"中的字符位置
str.find_last_of("aeiou");     // 查找最后一个出现在"aeiou"中的字符位置
str.find_first_not_of("aeiou");// 查找第一个不在"aeiou"中的字符位置
str.find_last_not_of("aeiou"); // 查找最后一个不在"aeiou"中的字符位置

// 使用示例
if (str.find("World") != string::npos) {
    cout << "找到了" << endl;
}

// 在指定区间[start, final)中查找
string str = "Hello World Hello C++";
size_t start = 6;      // 起始位置
size_t final = 11;     // 结束位置（不包含）

// 方法一：先截取子串再查找
string sub = str.substr(start, final - start);  // "World"
size_t pos = sub.find("or");  // 在子串中查找

// 方法二：在指定范围内查找（使用find的第二个参数）
size_t pos = str.find("or", start);  // 从start位置开始查找
if (pos != string::npos && pos < final) {
    // 找到了，且位置在[start, final)范围内
    cout << "在范围内找到了，位置：" << pos << endl;
}

// 方法三：使用迭代器在指定范围内查找
auto it = find(str.begin() + start, str.begin() + final, 'o');
if (it != str.begin() + final) {
    // 找到了字符'o'
    size_t pos = it - str.begin();  // 计算实际位置
    cout << "找到了，位置：" << pos << endl;
}

// 方法四：在指定范围内查找子串（完整函数）
size_t findInRange(const string& str, const string& pattern, 
                   size_t start, size_t final) {
    size_t pos = str.find(pattern, start);
    if (pos != string::npos && pos + pattern.length() <= final) {
        return pos;  // 找到了，且完全在范围内
    }
    return string::npos;  // 未找到或超出范围
}

// 使用示例
string text = "Hello World Hello C++";
size_t pos = findInRange(text, "World", 0, 11);  // 在[0,11)中查找
if (pos != string::npos) {
    cout << "找到了，位置：" << pos << endl;
}
```

### 字符串分割

```cpp
// 方法一：使用find和substr
vector<string> split(string s, char delimiter) {
    vector<string> result;
    size_t pos = 0;
    while ((pos = s.find(delimiter)) != string::npos) {
        result.push_back(s.substr(0, pos));
        s.erase(0, pos + 1);
    }
    result.push_back(s);
    return result;
}

// 方法二：使用stringstream
#include <sstream>
vector<string> split(string s, char delimiter) {
    vector<string> result;
    stringstream ss(s);
    string token;
    while (getline(ss, token, delimiter)) {
        result.push_back(token);
    }
    return result;
}

// 使用示例
string str = "apple,banana,orange";
vector<string> fruits = split(str, ',');
// fruits = {"apple", "banana", "orange"}
```
