## 6.浅拷贝与深拷贝
浅拷贝
    1.创建一个新对象，但对象内部的引用类型属性仍然指向原对象的相同内存地址
    2.只复制对象的第一层属性
    3.修改新对象的引用类型属性会影响原对象
深拷贝
    1.创建一个新对象，并递归复制所有嵌套的对象和数组
    2.确保新对象与原对象完全独立
    3.修改新对象不会影响原对象
```java
//浅拷贝
p2=p1;
//深拷贝
public Point copy() 
{
    Point newPoint = new Point();
    newPoint.x = this.x; 
    newPoint.y = this.y;  
    return newPoint;
}
Point p2 = p1.copy();
```

## 7.函数可变参数
| 方法声明 | 调用方式 | 编译结果 | 原因 |
|---------|---------|---------|------|
| `String... args` | `strscat("a", "b", "c")` | ✓ 通过 | 可变参数接受多个单独参数 |
| `String... args` | `strscat(new String[]{"a", "b"})` | ✓ 通过 | 可变参数也接受数组参数 |
| `String[] args` | `strscat(new String[]{"a", "b"})` | ✓ 通过 | 数组参数接受数组 |
| `String[] args` | `strscat("a", "b", "c")` | ✗ 失败 | 数组参数不接受多个单独参数 |
| `String... args` + `String[] args` | 任何调用 | ✗ 失败 | 方法重载冲突 |
```java
//实战演练：字符串连接
public static String strscat(String... args) {
    if (args == null || args.length == 0) {
        return "";
    }
    
    StringBuilder result = new StringBuilder();
    for (String arg : args) {
        if (arg != null) {
            result.append(arg);
        }
    }
    return result.toString();
}
```
```java
public static void main(String[] args) {
    // 测试多个字符串连接
    String result1 = strscat("Hello", " ", "World", "!");
    System.out.println("结果1: " + result1);
    
    // 测试中文字符串连接
    String result2 = strscat("Java", " ", "静态方法", " ", "示例");
    System.out.println("结果2: " + result2);
    
    // 测试单个字符串
    String result3 = strscat("单个字符串");
    System.out.println("结果3: " + result3);
    
    // 测试空参数
    String result4 = strscat();
    System.out.println("结果4(空参数): '" + result4 + "'");
    
    // 测试包含null的情况
    String result5 = strscat("Hello", null, "World");
    System.out.println("结果5(包含null): " + result5);
}
```
