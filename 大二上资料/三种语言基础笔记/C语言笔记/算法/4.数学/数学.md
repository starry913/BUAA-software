思路:利用归并排序

```c
#include <stdlib.h>
void mergeSort(int k[ ],int n);
void mSort(int k[], int tmp[], int left, int right);

            scanf("%d",&a[i]);

    }

void mergeSort(int k[ ],int n)//二路归并1 
{
    int *tmp;
    tmp = (int *)malloc(sizeof(int) * n);
    if(tmp != NULL) 
    {
        mSort(k, tmp, 0, n-1);
        free(tmp);
    } 
    else
        printf("No space for tmp array!!!\n");
}
void mSort(int k[], int tmp[], int left, int right)//二路归并2
{
    int center;
    if(left < right)
    {
        center = (left+right)/2;
        mSort(k, tmp, left, center);
        mSort(k, tmp, center+1, right);
        merge(k, tmp, left,center, right);
    }
}
void merge(int x[ ],int tmp[ ],int left,int leftend,int rightend)//二路归并3
{     
    int i=left, j=leftend+1, q=left;
    while(i<=leftend && j<=rightend)
    {
        if(x[i]<=x[j]) 
            tmp[q++]=x[i++];
        else
        {
            tmp[q++]=x[j++];
            count += leftend - i + 1;//核心代码
        }
    }    
    while(i<=leftend)
        tmp[q++]=x[i++];
    while(j<=rightend)
        tmp[q++]=x[j++];
    for(i=left; i<=rightend; i++)
        x[i]=tmp[i];
}
```

## 2. 快速傅里叶变换(FFT)

### 2.1 基本概念

**什么是FFT？**
- **DFT（离散傅里叶变换）**: 将n个点的时域信号转换为n个点的频域信号
- **FFT（快速傅里叶变换）**: 用分治思想快速计算DFT，时间复杂度从O(n²)降到O(n log n)
- **核心应用**: 多项式乘法、大整数乘法、卷积运算、信号处理

**为什么需要FFT？**
传统多项式乘法：两个n次多项式相乘需要O(n²)时间
使用FFT：可以降到O(n log n)，当n很大时优势明显

### 2.2 从多项式乘法理解FFT

**问题**: 计算两个多项式 A(x) 和 B(x) 的乘积 C(x) = A(x) × B(x)

**传统方法（系数表示法）**:
- A(x) = a₀ + a₁x + a₂x² + ... + aₙxⁿ
- B(x) = b₀ + b₁x + b₂x² + ... + bₙxⁿ
- 计算每个系数需要O(n²)时间

**FFT方法（点值表示法）**:
1. 将多项式转换为点值表示（FFT）：O(n log n)
2. 点值相乘：O(n)
3. 将点值转换回系数表示（IFFT）：O(n log n)
总时间：O(n log n)

### 2.2.1 FFT的本质：为什么既能做多项式乘法，也能做其他题？

**关键理解：FFT的核心功能是什么？**

FFT的本质是：**快速计算多项式在所有单位根处的值**

具体来说：
- **输入**：多项式的系数表示 [a₀, a₁, a₂, ..., aₙ₋₁]
- **输出**：多项式在单位根处的值 [A(ωₙ⁰), A(ωₙ¹), A(ωₙ²), ..., A(ωₙⁿ⁻¹)]

**为什么多项式乘法用FFT？**

多项式乘法的步骤：
1. **FFT**: 计算 A(x) 和 B(x) 在所有单位根处的值
   - 得到：A(ωₙ⁰), A(ωₙ¹), ..., A(ωₙⁿ⁻¹)
   - 得到：B(ωₙ⁰), B(ωₙ¹), ..., B(ωₙⁿ⁻¹)
2. **点值相乘**: C(ωₙᵏ) = A(ωₙᵏ) × B(ωₙᵏ)
3. **IFFT**: 将点值转换回系数表示

**为什么这道题也能用FFT？**

题目要求：z = Σ_{k=0}^{2^n-1} cos(k) · f(ω^k)

关键观察：
- **f(ω^k) 是什么？** 就是多项式 f(x) 在单位根 ω^k 处的值！
- **FFT正好计算这个！** FFT的输出就是 f(ω⁰), f(ω¹), f(ω²), ..., f(ω^(2^n-1))
- **不需要IFFT**：我们只需要点值，不需要转回系数

**总结对比**：

| 应用场景 | 使用FFT做什么 | 是否需要IFFT |
|---------|-------------|-------------|
| 多项式乘法 | 计算两个多项式在单位根处的值，然后相乘 | ✅ 需要（转回系数） |
| 这道题 | 计算多项式在单位根处的值，然后加权求和 | ❌ 不需要（直接用点值） |
| 信号处理 | 计算信号在频域的值 | ❌ 不需要（频域就是结果） |

**核心思想**：
- FFT = 系数表示 → 点值表示（在单位根处）
- IFFT = 点值表示 → 系数表示
- 多项式乘法需要：系数→点值→点值相乘→系数（需要IFFT）
- 这道题只需要：系数→点值（不需要IFFT）

**类比理解**：
- FFT就像"翻译器"：把多项式从"系数语言"翻译成"点值语言"
- 多项式乘法：翻译→计算→翻译回来
- 这道题：翻译→直接使用（不需要翻译回来）

### 2.3 单位根（核心数学工具）

**n次单位根**: 满足 ωⁿ = 1 的复数ω

**主n次单位根**: ωₙ = e^(2πi/n) = cos(2π/n) + i·sin(2π/n)

**重要性质**:
1. **周期性**: ωₙ^(k+n) = ωₙᵏ
2. **对称性**: ωₙ^(k+n/2) = -ωₙᵏ  （当n为偶数时）
3. **消去引理**: ωₙᵏ = ωₙₖᵏₖ（n是kn的约数）
4. **折半引理**: (ωₙ)² = ωₙ/₂

**可视化理解**:
- 单位根在复平面上均匀分布在单位圆上
- 例如4次单位根：ω₄⁰=1, ω₄¹=i, ω₄²=-1, ω₄³=-i
- 它们形成一个正n边形

### 2.4 FFT的分治思想

**DFT的定义**:
对于多项式 A(x) = a₀ + a₁x + ... + aₙ₋₁xⁿ⁻¹
DFT计算：yₖ = A(ωₙᵏ) = Σⱼ₌₀ⁿ⁻¹ aⱼ·(ωₙᵏ)ʲ

**分治策略**:
将多项式按奇偶项分成两部分：
- A₀(x) = a₀ + a₂x + a₄x² + ...  （偶数项）
- A₁(x) = a₁ + a₃x + a₅x² + ...  （奇数项）
- A(x) = A₀(x²) + x·A₁(x²)

**关键观察**:
- 计算A(ωₙᵏ)时，需要A₀(ωₙ²ᵏ)和A₁(ωₙ²ᵏ)
- 由于ωₙ² = ωₙ/₂，问题规模减半！
- 递归计算两个n/2点的DFT，然后合并

**合并过程（蝶形运算）**:
```
A(ωₙᵏ) = A₀(ωₙ/₂ᵏ) + ωₙᵏ·A₁(ωₙ/₂ᵏ)
A(ωₙ^(k+n/2)) = A₀(ωₙ/₂ᵏ) - ωₙᵏ·A₁(ωₙ/₂ᵏ)
```
注意：第二个公式利用了对称性 ωₙ^(k+n/2) = -ωₙᵏ

### 2.5 FFT实现（详细注释版）

```c
#include <stdio.h>
#include <stdlib.h>
#include <math.h>
#include <string.h>

#define PI 3.14159265358979323846

// 复数结构体
typedef struct {
    double real;
    double imag;
} Complex;

// 复数加法
Complex add(Complex a, Complex b) {
    Complex c;
    c.real = a.real + b.real;
    c.imag = a.imag + b.imag;
    return c;
}

// 复数减法
Complex sub(Complex a, Complex b) {
    Complex c;
    c.real = a.real - b.real;
    c.imag = a.imag - b.imag;
    return c;
}

// 复数乘法
Complex mul(Complex a, Complex b) {
    Complex c;
    c.real = a.real * b.real - a.imag * b.imag;
    c.imag = a.real * b.imag + a.imag * b.real;
    return c;
}

// 位逆序置换
void bitReverse(Complex *a, int n) {
    int i, j, k;
    for (i = 1, j = n >> 1; i < n - 1; i++) {
        if (i < j) {
            Complex temp = a[i];
            a[i] = a[j];
            a[j] = temp;
        }
        k = n >> 1;
        while (j >= k) {
            j -= k;
            k >>= 1;
        }
        j += k;
    }
}

// FFT主函数
void fft(Complex *a, int n, int inv) {
    bitReverse(a, n);
    int i, j, k;
    for (int len = 2; len <= n; len <<= 1) {
        double ang = 2 * PI / len * (inv ? -1 : 1);
        Complex wlen;
        wlen.real = cos(ang);
        wlen.imag = sin(ang);
        for (i = 0; i < n; i += len) {
            Complex w;
            w.real = 1.0;
            w.imag = 0.0;
            for (j = 0; j < len / 2; j++) {
                Complex u = a[i + j];
                Complex v = mul(a[i + j + len / 2], w);
                a[i + j] = add(u, v);
                a[i + j + len / 2] = sub(u, v);
                w = mul(w, wlen);
            }
        }
    }
    if (inv) {
        for (i = 0; i < n; i++) {
            a[i].real /= n;
            a[i].imag /= n;
        }
    }
}

// 多项式乘法：计算 a * b，结果存入 c
void polyMultiply(int *a, int n1, int *b, int n2, int *c) {
    int n = 1;
    while (n < n1 + n2 - 1) n <<= 1;
    
    Complex *fa = (Complex*)malloc(n * sizeof(Complex));
    Complex *fb = (Complex*)malloc(n * sizeof(Complex));
    
    // 初始化
    for (int i = 0; i < n; i++) {
        fa[i].real = (i < n1) ? a[i] : 0;
        fa[i].imag = 0;
        fb[i].real = (i < n2) ? b[i] : 0;
        fb[i].imag = 0;
    }
    
    // FFT
    fft(fa, n, 0);
    fft(fb, n, 0);
    
    // 点值相乘
    for (int i = 0; i < n; i++) {
        fa[i] = mul(fa[i], fb[i]);
    }
    
    // IFFT
    fft(fa, n, 1);
    
    // 提取结果
    for (int i = 0; i < n1 + n2 - 1; i++) {
        c[i] = (int)(fa[i].real + 0.5);
    }
    
    free(fa);
    free(fb);
}

// 示例：多项式乘法
int main() {
    // 示例：计算 (1 + 2x + 3x^2) * (4 + 5x)
    // 结果应该是：4 + 13x + 22x^2 + 15x^3
    
    int a[] = {1, 2, 3};  // 1 + 2x + 3x^2
    int b[] = {4, 5};      // 4 + 5x
    int c[10];
    
    polyMultiply(a, 3, b, 2, c);
    
    printf("多项式乘法结果：\n");
    for (int i = 0; i < 4; i++) {
        printf("x^%d 的系数: %d\n", i, c[i]);
    }
    // 输出：4, 13, 22, 15
    
    return 0;
}
```

### 关键点说明

1. **位逆序置换**: FFT需要将数组按位逆序排列，这是分治的基础
2. **蝶形运算**: 每一层将两个子问题的结果合并
3. **单位根**: 使用欧拉公式 e^(iθ) = cos(θ) + i*sin(θ) 计算单位根
4. **IFFT**: 逆FFT只需在最后除以n，并将单位根取负

### 时间复杂度
- **FFT**: O(n log n)
- **多项式乘法**: O(n log n)，比传统O(n²)快得多

### 注意事项
- n必须是2的幂次，不足需要补0
- 结果可能有精度误差，需要四舍五入
- 对于大整数乘法，可以用FFT加速

### 2.6 例题：计算多项式在单位根处的加权和

**题目描述**:
- 给定一个 2^n-1 次整系数多项式 f(x) = Σ_{k=0}^{2^n-1} c_k x^k
- 定义复数 ω = cos(2π/2^n) + i·sin(2π/2^n)（2^n次单位根）
- 计算 z = Σ_{k=0}^{2^n-1} cos(k) · f(ω^k)

**解题思路**:
1. **理解问题**: f(ω^k) 就是多项式在单位根处的值，这正是FFT要计算的！
2. **使用FFT**: 用FFT计算 f(ω^0), f(ω^1), ..., f(ω^(2^n-1))
3. **加权求和**: 对每个 f(ω^k) 乘以 cos(k)，然后求和

**关键点**:
- FFT计算的就是多项式在所有单位根处的值（DFT）
- 题目中的多项式有 2^n 个系数，正好可以用 2^n 点FFT
- 不需要IFFT，因为我们只需要点值，不需要转回系数

**完整代码**:
```c
#include <stdio.h>
#include <stdlib.h>
#include <math.h>

#define PI 3.14159265358979323846

// 复数结构体
typedef struct {
    double real;
    double imag;
} Complex;

// 复数加法
Complex add(Complex a, Complex b) {
    Complex c;
    c.real = a.real + b.real;
    c.imag = a.imag + b.imag;
    return c;
}

// 复数减法
Complex sub(Complex a, Complex b) {
    Complex c;
    c.real = a.real - b.real;
    c.imag = a.imag - b.imag;
    return c;
}

// 复数乘法
Complex mul(Complex a, Complex b) {
    Complex c;
    c.real = a.real * b.real - a.imag * b.imag;
    c.imag = a.real * b.imag + a.imag * b.real;
    return c;
}

// 位逆序置换
void bitReverse(Complex *a, int n) {
    int i, j, k;
    for (i = 1, j = n >> 1; i < n - 1; i++) {
        if (i < j) {
            Complex temp = a[i];
            a[i] = a[j];
            a[j] = temp;
        }
        k = n >> 1;
        while (j >= k) {
            j -= k;
            k >>= 1;
        }
        j += k;
    }
}

// FFT主函数
// inv=0: 正变换(DFT), inv=1: 逆变换(IDFT)
void fft(Complex *a, int n, int inv) {
    bitReverse(a, n);
    int i, j;
    for (int len = 2; len <= n; len <<= 1) {
        // 计算单位根的角度
        double ang = 2 * PI / len * (inv ? -1 : 1);
        Complex wlen;
        wlen.real = cos(ang);
        wlen.imag = sin(ang);
        
        // 对每个长度为len的块进行蝶形运算
        for (i = 0; i < n; i += len) {
            Complex w;
            w.real = 1.0;
            w.imag = 0.0;
            
            // 蝶形运算
            for (j = 0; j < len / 2; j++) {
                Complex u = a[i + j];
                Complex v = mul(a[i + j + len / 2], w);
                a[i + j] = add(u, v);
                a[i + j + len / 2] = sub(u, v);
                w = mul(w, wlen);  // 更新单位根
            }
        }
    }
    
    // 如果是逆变换，需要除以n
    if (inv) {
        for (i = 0; i < n; i++) {
            a[i].real /= n;
            a[i].imag /= n;
        }
    }
}

int main() {
    int n;
    scanf("%d", &n);
    
    int size = 1 << n;  // 2^n
    int *coeff = (int*)malloc(size * sizeof(int));
    
    // 读入多项式系数
    for (int i = 0; i < size; i++) {
        scanf("%d", &coeff[i]);
    }
    
    // 将系数转换为复数数组
    Complex *a = (Complex*)malloc(size * sizeof(Complex));
    for (int i = 0; i < size; i++) {
        a[i].real = coeff[i];
        a[i].imag = 0.0;
    }
    
    // 使用FFT计算多项式在所有单位根处的值
    // fft后，a[k] 就是 f(ω^k) 的值
    fft(a, size, 0);
    
    // 计算 z = Σ_{k=0}^{2^n-1} cos(k) · f(ω^k)
    Complex z;
    z.real = 0.0;
    z.imag = 0.0;
    
    for (int k = 0; k < size; k++) {
        // cos(k) 是实数，所以 cos(k) · f(ω^k) = cos(k) · (real + i·imag)
        double cos_k = cos(k);
        Complex term;
        term.real = cos_k * a[k].real;
        term.imag = cos_k * a[k].imag;
        z = add(z, term);
    }
    
    // 输出结果，保留两位小数
    printf("%.2f %.2f\n", z.real, z.imag);
    
    free(coeff);
    free(a);
    return 0;
}
```

**代码解释**:
1. **读入数据**: n 和 2^n 个系数
2. **FFT变换**: 将系数表示转换为点值表示，a[k] 存储 f(ω^k)
3. **加权求和**: 对每个 f(ω^k) 乘以 cos(k) 后累加
4. **输出**: 输出复数 z 的实部和虚部

**时间复杂度**: O(n·2^n) = O(N log N)，其中 N = 2^n

**关键理解**:
- FFT的本质就是计算多项式在单位根处的值
- 这道题直接利用了FFT的这个特性，不需要额外的转换
- cos(k) 是实数权重，直接与复数值相乘即可

### 2.7 FFT通用板子（封装版）

**使用说明**：这是一个封装好的FFT板子，每个函数都有清晰的输入输出说明，适用于大多数FFT相关题目。

```c
#include <stdio.h>
#include <stdlib.h>
#include <math.h>

#define PI 3.14159265358979323846

// ==================== 数据结构 ====================
// 复数结构体
typedef struct {
    double real;  // 实部
    double imag;  // 虚部
} Complex;

// ==================== 复数基本运算 ====================
// 这些函数不需要理解原理，直接使用即可

// 复数加法
Complex cadd(Complex a, Complex b) {
    Complex c;
    c.real = a.real + b.real;
    c.imag = a.imag + b.imag;
    return c;
}

// 复数减法
Complex csub(Complex a, Complex b) {
    Complex c;
    c.real = a.real - b.real;
    c.imag = a.imag - b.imag;
    return c;
}

// 复数乘法
Complex cmul(Complex a, Complex b) {
    Complex c;
    c.real = a.real * b.real - a.imag * b.imag;
    c.imag = a.real * b.imag + a.imag * b.real;
    return c;
}

// ==================== FFT核心函数 ====================

/**
 * 位逆序置换（内部函数，FFT需要）
 * 输入: a[] - 复数数组, n - 数组长度（必须是2的幂）
 * 输出: 原地修改数组，按位逆序排列
 * 用途: FFT算法的预处理步骤，不需要单独调用
 */
void bitReverse(Complex *a, int n) {
    int i, j, k;
    for (i = 1, j = n >> 1; i < n - 1; i++) {
        if (i < j) {
            Complex temp = a[i];
            a[i] = a[j];
            a[j] = temp;
        }
        k = n >> 1;
        while (j >= k) {
            j -= k;
            k >>= 1;
        }
        j += k;
    }
}

/**
 * FFT主函数（核心函数）
 * 输入: 
 *   - a[]: 复数数组，存储多项式的系数（实部）或点值（结果）
 *   - n: 数组长度，必须是2的幂（如2, 4, 8, 16, ...）
 *   - inv: 0表示正变换(FFT)，1表示逆变换(IFFT)
 * 输出: 
 *   - 原地修改数组a[]
 *   - 如果inv=0: a[k] = 多项式在单位根ω^k处的值
 *   - 如果inv=1: a[k] = 多项式的第k个系数（需要先做FFT，再点值相乘）
 * 用途: 
 *   - 正变换：将系数表示转换为点值表示
 *   - 逆变换：将点值表示转换为系数表示
 * 
 * 使用示例:
 *   Complex a[8];
 *   // 初始化a[]为多项式系数
 *   fft(a, 8, 0);  // 正变换，得到点值
 *   // 现在a[k] = 多项式在ω^k处的值
 */
void fft(Complex *a, int n, int inv) {
    bitReverse(a, n);
    int i, j;
    for (int len = 2; len <= n; len <<= 1) {
        double ang = 2 * PI / len * (inv ? -1 : 1);
        Complex wlen;
        wlen.real = cos(ang);
        wlen.imag = sin(ang);
        for (i = 0; i < n; i += len) {
            Complex w;
            w.real = 1.0;
            w.imag = 0.0;
            for (j = 0; j < len / 2; j++) {
                Complex u = a[i + j];
                Complex v = cmul(a[i + j + len / 2], w);
                a[i + j] = cadd(u, v);
                a[i + j + len / 2] = csub(u, v);
                w = cmul(w, wlen);
            }
        }
    }
    if (inv) {
        for (i = 0; i < n; i++) {
            a[i].real /= n;
            a[i].imag /= n;
        }
    }
}

/**
 * 计算两个多项式的乘积
 * 输入:
 *   - a[]: 第一个多项式的系数（整数数组）
 *   - n1: 第一个多项式的项数（次数+1）
 *   - b[]: 第二个多项式的系数（整数数组）
 *   - n2: 第二个多项式的项数（次数+1）
 *   - c[]: 输出数组，存储结果多项式的系数
 * 输出:
 *   - c[]: 乘积多项式的系数，c[i]是x^i的系数
 *   - 结果多项式的项数 = n1 + n2 - 1
 * 用途: 计算两个多项式的乘积，时间复杂度O(n log n)
 * 
 * 使用示例:
 *   int a[] = {1, 2, 3};  // 1 + 2x + 3x^2
 *   int b[] = {4, 5};      // 4 + 5x
 *   int c[10];
 *   polyMultiply(a, 3, b, 2, c);
 *   // 结果: c = {4, 13, 22, 15}  // 4 + 13x + 22x^2 + 15x^3
 */
void polyMultiply(int *a, int n1, int *b, int n2, int *c) {
    // 计算需要的FFT长度（必须是2的幂）
    int n = 1;
    while (n < n1 + n2 - 1) n <<= 1;
    
    // 分配内存
    Complex *fa = (Complex*)malloc(n * sizeof(Complex));
    Complex *fb = (Complex*)malloc(n * sizeof(Complex));
    
    // 初始化：将整数系数转换为复数
    for (int i = 0; i < n; i++) {
        fa[i].real = (i < n1) ? a[i] : 0;
        fa[i].imag = 0;
        fb[i].real = (i < n2) ? b[i] : 0;
        fb[i].imag = 0;
    }
    
    // 步骤1: 对两个多项式做FFT（系数→点值）
    fft(fa, n, 0);
    fft(fb, n, 0);
    
    // 步骤2: 点值相乘
    for (int i = 0; i < n; i++) {
        fa[i] = cmul(fa[i], fb[i]);
    }
    
    // 步骤3: IFFT（点值→系数）
    fft(fa, n, 1);
    
    // 步骤4: 提取结果（四舍五入）
    for (int i = 0; i < n1 + n2 - 1; i++) {
        c[i] = (int)(fa[i].real + 0.5);
    }
    
    free(fa);
    free(fb);
}

/**
 * 计算多项式在单位根处的值（DFT）
 * 输入:
 *   - coeff[]: 多项式的系数（整数数组）
 *   - n_coeff: 系数的个数
 *   - result[]: 输出数组，存储结果
 *   - n_result: 结果数组长度（必须是2的幂，且>=n_coeff）
 * 输出:
 *   - result[k] = 多项式在单位根ω^k处的值（复数）
 * 用途: 计算多项式在所有单位根处的值，用于需要点值的题目
 * 
 * 使用示例:
 *   int coeff[] = {1, 2, 3, 4};
 *   Complex result[4];
 *   polyDFT(coeff, 4, result, 4);
 *   // 现在result[k] = 多项式在ω^k处的值
 */
void polyDFT(int *coeff, int n_coeff, Complex *result, int n_result) {
    // 初始化：将系数转换为复数
    for (int i = 0; i < n_result; i++) {
        result[i].real = (i < n_coeff) ? coeff[i] : 0;
        result[i].imag = 0;
    }
    
    // FFT：系数→点值
    fft(result, n_result, 0);
    
    // 现在result[k]就是多项式在ω^k处的值
}

// ==================== 使用示例 ====================

// 示例1: 多项式乘法
void example1_polyMultiply() {
    int a[] = {1, 2, 3};  // 1 + 2x + 3x^2
    int b[] = {4, 5};      // 4 + 5x
    int c[10];
    
    polyMultiply(a, 3, b, 2, c);
    
    printf("多项式乘法结果：\n");
    for (int i = 0; i < 4; i++) {
        printf("x^%d 的系数: %d\n", i, c[i]);
    }
    // 输出: 4, 13, 22, 15
}

// 示例2: 计算多项式在单位根处的值
void example2_polyDFT() {
    int coeff[] = {1, 2, 3, 4};
    Complex result[4];
    
    polyDFT(coeff, 4, result, 4);
    
    printf("多项式在单位根处的值：\n");
    for (int i = 0; i < 4; i++) {
        printf("f(ω^%d) = %.2f + %.2fi\n", 
               i, result[i].real, result[i].imag);
    }
}

// 示例3: 解决"计算加权和"类型的题目
void example3_weightedSum() {
    int n;
    scanf("%d", &n);
    int size = 1 << n;  // 2^n
    
    int *coeff = (int*)malloc(size * sizeof(int));
    for (int i = 0; i < size; i++) {
        scanf("%d", &coeff[i]);
    }
    
    // 计算多项式在单位根处的值
    Complex *values = (Complex*)malloc(size * sizeof(Complex));
    polyDFT(coeff, size, values, size);
    
    // 计算加权和 z = Σ cos(k) · f(ω^k)
    Complex z;
    z.real = 0.0;
    z.imag = 0.0;
    for (int k = 0; k < size; k++) {
        double cos_k = cos(k);
        Complex term;
        term.real = cos_k * values[k].real;
        term.imag = cos_k * values[k].imag;
        z = cadd(z, term);
    }
    
    printf("%.2f %.2f\n", z.real, z.imag);
    
    free(coeff);
    free(values);
}
```

### 板子使用指南

**1. 什么时候用哪个函数？**

| 题目类型 | 使用的函数 | 说明 |
|---------|----------|------|
| 多项式乘法 | `polyMultiply()` | 直接调用，输入两个多项式系数，输出乘积系数 |
| 需要点值 | `polyDFT()` | 计算多项式在单位根处的值 |
| 自定义操作 | `fft()` | 需要自己处理点值相乘等操作 |

**2. 常见问题**

**Q: n必须是2的幂吗？**
A: 是的。如果多项式有m个系数，需要找到最小的2的幂n，使得n≥m，然后补0。

**Q: 如何补0？**
A: 在调用FFT前，将数组后面补0即可。`polyMultiply()`和`polyDFT()`已经自动处理了。

**Q: 精度问题？**
A: 结果可能有浮点误差，使用`(int)(x + 0.5)`四舍五入。

**Q: 内存管理？**
A: `polyMultiply()`和`polyDFT()`内部会分配内存，使用完后会自动释放。如果直接使用`fft()`，需要自己管理内存。
