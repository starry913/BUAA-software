## 1.指针
### 1.1 使用任何指针变量之前必须先给它赋一个所指合法具体对象的地址值
```c
//指针没有实体地址的错误
    int x = 1;
    int *px;
    *px = x;//错误

	char *string;
    scanf(“%s”, string);//错误
    strcpy(string, “Hello”);

    char a[]=“hello”, *p=“hello”;
    a[0] = ‘b’;   	/*正确 */
    *p = ‘b’;  /*错误，不能修改常量值*/

```
### 1.2 动态内存管理
```c
    #include <stdlib.h>
    char *s; 
    int *intptr;
    s = (char *)malloc(32); /* s指向大小为32个字节（字符）的空间*/
    s = (char *)malloc(strlen(p)+1);/* s指向能正好存放字符串p的空间*/
    intptr = (int *)malloc(sizeof(int)*10);/* ptr指向能存放10个整型元素的空间*/
    free(s);
    free(intptr);
```
### 1.3 常见考点
```c
(1)  (*px)++; //先取px所指对象内容进行运算，然后对其加1
(2)  *px++;
(3)  *(px++);// 2、3等价，先取px所指对象内容进行运算，然后指针px加1
```

## 2.结构体
### 2.1 结构体定义
```c
1）无结构名
    struct 
    {
    …
    } s1, s2;
一般适用于说明本地变量。
2）有结构名
    struct date 
    {
    …
    } ;
    struct date s1, s2;
通常用来说明外部结构变量，或需要在多个函数中用到的相同的结构的变量。
3）使用typedef
    typedef struct 
    {
    …
    } DATE;
则变量定义为：DATE d, *pd, ad[10];
```

### 2.2 结构体使用
```c
struct  date  *pd, d; （pd是指向struct  date的指针）
pd = &d	（指向一个已有结构变量）
pd = (struct date *) malloc(sizeof(struct date));

pd->year = 1958;            //pd->year与(*pd).year 是完全等价的
strcpy(pd->mon_name, “OCT”);

emp.birthdate.year  = 1970;
emp.birthdate.month = 8;
```
