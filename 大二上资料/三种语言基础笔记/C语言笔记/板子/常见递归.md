# 常见递归
## 1.组合数 $C_m^n$
```c
int c(int m,int n)
{
	if(n==0) return 1;
	else if(m==n) return 1;
	else if(m<n)  return 0;
	else if(n==1) return m;
	return c(m-1,n)+c(m-1,n-1);
}
```

## 2.1汉诺塔

void hanoi(int n, char from, char via, char to) //n个片子，从小到大放，从A棒移到c棒
{
    if(n == 1) move(n, from, to);
    else
    {
        hanoi(n-1, from, to, via);
        move(n, from, to);
        hanoi(n-1, via, from, to);
    }
}

## 3.⼆分法求解函数⽅程
```c
对于单调递增函数：
/*
* ⼆分法求 func(x) = y 的解
*
* @param left 有解区间左端点
* @param right 有解区间右端点
* @param y     函数值
* @param eps   允许误差
* @return     令 func(x) = y 的 x 的近似值
*/
double solve(double left, double right, double y, double eps) {
   double mid;
   while (right - left > eps) {
       mid = (right + left) / 2;
       if (func(mid) > y)        // 递减则 func(mid) < y
           right = mid;
       else left = mid;
  }
   return mid;
}
```
