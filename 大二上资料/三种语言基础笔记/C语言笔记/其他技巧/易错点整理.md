## 归纳
### 1.数据类型
1. **a/b** 当a、b都为整型时，表示整除<br>
            当a、b有浮点型时，表示正常除法
2. **scanf(" %c",&a);** %c前加空格表示跳过前导空白(视情况，一般都可以用)
3. 常数类型
```c
1 2 3                      // 不加任何后缀，是 int 类型
1.0 2.0                    // 加⼩数点，是 double 类型
1LL 2LL                    // 加 LL 是 long long 类型
1U 2U                      // 加 U 是 unsigned int 类型
1ULL 2ULL                  // 加 ULL 是 unsigned long long 类型
double a = 3 / 2;          // 这样写 b 的值为 1.000000
double a = 3.0 / 2;        // b 的值为 1.500000
long long a = 1 << 40;     // 错误，会在赋值号右侧发⽣溢出
long long a = 1LL << 40;   // 正确
```
4. double型判断（E6程设助教大战高代）
```c
#include <math.h>
#define eps 1e-6
if(fabs(a)>=eps)   //if(a!=0)
if(fabs(a)<eps)    //if(a==0)
if(a-b>eps)        //if(a>b)
if(a-b<-eps)       //if(a<b)
if(fabs(a-b)<eps)  //if(a==b)
```

5. int+int 会超出int 范围(小宇的AT+A)<br>
此时要用long long

6. 取余
若涉及到减法取余，如(a-b)%c<br>
要加余数再取余，避免余数为负数，(a-b+c)%c

7. 位运算
   位运算的运算优先级很复杂！一定要多加括号<br>
   如x=(a^b)
