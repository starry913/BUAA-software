# 时序逻辑电路
## 1.时序逻辑电路概念
![alt text](360截图20251010155916683.jpg)
触发器（Flip-Flop，FF）就是构成记忆功能部件的基本单元，或者说是实现存储（记忆）功能的基本单元电路。
![alt text](360截图20251010160134495.jpg)
触发器由锁存器构成
| **特性**    | **锁存器（Latch）** | **触发器（Flip-Flop）** |
| --------- | -------------- | ------------------ |
| **触发方式**  | 电平敏感（高/低电平）    | 边沿敏感（上升/下降沿）       |
| **时钟依赖**  | 无              | 有（同步时钟）            |
| **透明性**   | 使能期间透明（输入=输出）  | 非透明（仅边沿采样）         |
| **抗干扰能力** | 弱（易受毛刺影响）      | 强（仅边沿采样）           |
| **典型结构**  | 单个锁存器          | 两个锁存器（主从结构）        |
| **应用场景**  | 异步电路、简单存储      | 同步时序电路（如寄存器）       |

### 2.2 钟控RS锁存器
在数字系统中，为了协调各部分电路的运行，常常要求某些锁存器在时钟信号的控制下同时动作，这就需要增加一个控制端（时钟），只有在控制端作用脉冲时锁存器才能动作。
![alt text](360截图20251010161824556.jpg)
![alt text](360截图20251010162435083.jpg)

### 2.3 钟控D锁存器
因为S、R不能同时为1，为了不使这种情况出现，将钟控RS锁存器的输入由R、S双端输入改为单端输入（D）
![alt text](360截图20251010163222544.jpg)
![alt text](360截图20251010163321317.jpg)
当CP＝1时，若D=0，相当于S=0，R=1，锁存器置“0”
若D=1，相当于S=1，R=0，锁存器置“1”。
当CP＝0时，输出不变。

## 5.有限状态机
有限状态机（Finite State Machine，FSM）是表示有限个状态以及这些状态之间的转移和动作（输入）之间关系的离散数学模型。
有限状态机是组合逻辑和寄存器逻辑的特殊组合。组合逻辑部分包括次态逻辑和输出逻辑，分别用于状态译码和产生输出信号；寄存器逻辑部分用于存储状态。

1. 摩尔(Moore)型状态机--输出信号仅与当前状态有关
2. 米里(Mealy)型状态机--输出信号与当前状态及输入信号有关
